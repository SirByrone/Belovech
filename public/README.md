# Image Assets Folder

This is where you should place all your image files for the Belovech website.

## 📁 Folder Structure

```
public/
├── logo.png                    # Your main logo (golden crescent design)
├── favicon.ico                 # Website favicon
└── images/                     # Additional images folder
    ├── team/                   # Team member photos
    ├── portfolio/              # Portfolio/project images
    ├── blog/                   # Blog post images
    └── icons/                  # Icon files
```

## 🖼️ Required Images

### 1. Logo Image ✅ ADDED
- **File**: `logo.jpg`
- **Size**: 200x200px (or higher resolution)
- **Format**: JPG with golden theme
- **Description**: Your golden crescent logo with quantum network pattern

## 📝 How to Add Your Images

1. **Copy your logo image** and rename it to `logo.png`
2. **Place the file** directly in the `public/` folder
3. **Refresh your browser** to see the changes

## 🔧 Image Optimization Tips

- **Logo**: Keep file size under 50KB for fast loading
- **Formats**: PNG for logos with transparency, JPG for photos
- **Dimensions**: Use high-resolution images (Next.js will optimize them automatically)

## 📍 File Paths in Code

- Logo: `/logo.png`
- Favicon: `/favicon.ico`

The website will automatically use these images once you place them in the public folder!
